# Code of Conduct

This repository uses a short placeholder code of conduct for demo purposes.

## Our Standard

We want this project to feel respectful, constructive, and welcoming to everyone who participates.

## Expected Behavior

- Be considerate in discussions and reviews.
- Assume good intent and give actionable feedback.
- Keep communication professional and inclusive.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Repeated disruptive behavior in discussions or reviews.
- Sharing private information without permission.

## Reporting

If an issue comes up, contact the project maintainers through the usual repository communication channel. For a real project, replace this section with a specific reporting process.
